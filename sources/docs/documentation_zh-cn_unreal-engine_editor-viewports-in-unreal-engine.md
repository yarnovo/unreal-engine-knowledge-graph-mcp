**视口（Viewports）** 是你在虚幻引擎中创建世界的窗口。你可以像在游戏中那样在视口中移动，也可以像建筑蓝图那样用于设计各种方案。虚幻编辑器的视口包含各种工具和可视化选项，能帮助你准确查看所需的数据。
