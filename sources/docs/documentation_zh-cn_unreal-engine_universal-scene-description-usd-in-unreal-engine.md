**通用场景描述** （USD）交换格式是由皮克斯公司（Pixar）开发的开源格式，用于稳健、可扩展地交换及增强任意3D场景，这些场景可能包含许多基础资产。USD不仅提供了多类工具用于读取、写入、编辑和快速预览3D几何图形和着色效果，还提供了元素资产（例如模型）或动画的交换。
