**StateTree** 是一种通用分层状态机，组合了行为树中的 **选择器（Selectors）** 与状态机中的 **状态（States）** 和 **过渡（Transitions）** 。使用StateTree，你可以创建非常高效、保持灵活且井然有序的逻辑。
