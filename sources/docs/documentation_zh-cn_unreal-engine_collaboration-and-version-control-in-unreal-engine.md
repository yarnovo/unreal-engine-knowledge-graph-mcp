虚幻引擎中整合了两种版本控制软件的完整功能，让大家在项目上能够协同工作，这两个软件是 Perforce 和 SVN。

版本控制软件能在团队的成员之间共享资源文件或代码文件，并能够提供存储备份以及文件的历史记录，能在某些文件发生问题时回滚到早先的可用版本上。
