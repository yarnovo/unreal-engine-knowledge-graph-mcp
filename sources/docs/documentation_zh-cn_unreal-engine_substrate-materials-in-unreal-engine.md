学习使用此**Beta**功能，但在发布产品中需要谨慎使用。

**Substrate** 是虚幻引擎5中的材质创建方法，它取代了原有的固定类型的着色模型和混合模式，例如"Default Lit" 和 "Clear Coat"，转而提供了一个更具表现力和模块化程度的框架。
