**物理材质（Physical Materials）** 用于定义物理对象在与世界场景动态交互时的响应。物理材质非常容易使用。创建一个物理材质将提供一组默认值，这与应用于所有物理对象的默认物理材质相同。示例为角色的无生命物体（布娃娃）、可移动大木箱等。
