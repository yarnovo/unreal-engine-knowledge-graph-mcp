Niagara为你提供了充分的灵活性，允许你创建自定义模块。你可以用Niagara脚本编辑器来定义你的逻辑流程，保存你的模块，然后在Niagara栈中应用它们。

## Niagara中的版本管理模块

当你需要经常创建新版本的模块时，最好启用版本管理功能，这样就能避免对使用老版本模块的场景造成破坏。本文详细探讨了这点。
