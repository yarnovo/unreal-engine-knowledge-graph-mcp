**舞台关卡（Stage Levels）** 文件夹包含 *同时* 拥有环境 *和* 舞台 **Actor** 的所有 **关卡资产**。需要使用nDisplay进行渲染时，可打开这些资产。舞台关卡按关卡资产中使用的舞台进行分类。

-   EpicLA
    
    -   CaveEntrance\_Epic\_LA - 持久关卡，带有所有子关卡和舞台拓扑
        
    -   SpaceJunkyard\_Epic\_LA
        
    -   CavePath\_EpicLA
