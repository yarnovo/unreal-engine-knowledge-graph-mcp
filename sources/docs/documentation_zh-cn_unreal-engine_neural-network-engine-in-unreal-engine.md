学习使用此**Beta**功能，但在发布产品中需要谨慎使用。

**神经网络引擎（NNE）** 提供了一个常用API，以访问不同的神经网络运行时，并在不需要运行时特定编码的情况下对神经网络求值。NNE有各种各样的用途，可以应用于实时推理，以使用人工智能（AI）强化游戏，并实基于编辑器的功能，如资产操作、查询和美术师辅助工具。
