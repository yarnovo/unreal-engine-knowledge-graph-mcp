-   概述Unity和虚幻引擎之间的异同。

-   学习如何将Unity资产导入到虚幻引擎中。
-   了解虚幻引擎的渲染技术。

如果你以前从未使用过虚幻引擎，请先阅读[Unity转虚幻引擎概述](/documentation/zh-cn/unreal-engine/unity-to-unreal-engine-overview)，该页面介绍了与Unity关键术语和概念相对应的虚幻引擎术语和概念。
