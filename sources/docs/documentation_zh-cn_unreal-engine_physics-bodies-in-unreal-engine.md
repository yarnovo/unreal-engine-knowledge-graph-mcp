代理形体使 3D 网格体的复杂程度不断增加，它必须用于 3D 程序中驱动物理模拟；物理对象（BodyInstances）是虚幻引擎 用于物理模拟的简单 3D 网格体。它们可以方块、球体、胶囊体或凸包的形态存在。
