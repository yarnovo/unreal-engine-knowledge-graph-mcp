**IK Rig** 系统提供了交互式创建 **解算器** 的方法，用于为你的骨骼网格体执行姿势编辑。然后，生成的IK Rig资产可以嵌入到任何动画系统中，例如动画蓝图，以便动态修改基于姿势的解算器参数。

此外，**IK重定向** 系统可以用于在不同比例的角色之间传输动画，无论是运行时还是离线创建新动画序列。

此页面包含文档链接，涵盖虚幻引擎的 **IK Rig** 和 **重定向** 工具，以及它们工作流程的实际示例。
