关卡编辑器包含 **网格体绘制（Mesh Paint）** 模式。这是一种在编辑器内直接将颜色和纹理应用到网格体的方法。你有多种方法可以将颜色数据应用到网格体。此种方法可以通过材质直接作用于网格体本身灭火时直接作用于纹理资产。这些绘制网格体的方法各有专用的绘制工具，并使颜色数据作用于网格体。

以下主题可以帮助你详细了解如何使用该模式并上手。
