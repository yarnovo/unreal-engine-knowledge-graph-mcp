在为头戴式显示设备（HMD）开发内容时，OpenXR立体图层（OpenXR Stereo Layers）允许你单独将某张纹理重新投射到某个单独的渲染通道中。这使得HMD中的UI内容可以单独渲染，从而不必受某些设置（比如后期处理、抗锯齿）的影响。
