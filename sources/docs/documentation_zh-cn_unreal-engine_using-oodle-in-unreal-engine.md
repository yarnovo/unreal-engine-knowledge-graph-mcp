**Oodle** 套件的跨平台数据压缩方案现已集成到 **虚幻引擎** 中。凭借Oodle，你压缩游戏下载包的大小，提升关卡加载速度，压缩网络数据包，从而改善玩家的体验。

虚幻引擎集成了三种Oodle解决方案。

-   **Oodle数据**
    
-   **Oodle网络**
    
-   **Oodle纹理**
    

所有Oodle解决方案都可以通过其插件单独启用或禁用。在虚幻引擎中，以上三种方案都会默认启用。

![The Oodle plugins](odle-plugins.png)(w:600)

有关各个Oodle解决方案的使用细节，请参考其单独页面。
