## 纹理导入

### 虚拟纹理

**部分**

**描述**

**自动虚拟纹理大小（Auto Virtual Texturing Size）**

在导入纹理时，为大于或者等于此大小的纹理，自动启动 "虚拟纹理流送（Virtually Texture Streaming）" 纹理设置，将其调整到该大小。

该设置不会影响项目中的现有纹理。
