**虚幻引擎** 中的 **Paper 2D** 是一种基于Sprite的系统，可用于在虚幻引擎中开发2D或2D/3D相结合的游戏。Paper 2D 的核心是 **Sprite**（它们是一种映射了纹理和相关材质的平面网格体）。在虚幻引擎中，你可以通过 **Sprite 编辑器** 对 Sprite 进行编辑，并利用 **Flipbooks** 实现基于 Sprite 的动画（通过设置关键帧和帧时长来实现基于Sprite的序列动画）。
