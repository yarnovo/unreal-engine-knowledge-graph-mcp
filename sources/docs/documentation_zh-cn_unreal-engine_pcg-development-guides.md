**程序化内容生成框架（Procedural Content Generation Framework (PCG)）** 是用于在虚幻引擎内创建你自己的程序化内容的工具集。该框架专为可扩展性和交互性而构建，可轻松集成到现有世界构建管线中，实际上模糊了程序化和更传统的工作流程之间的界限。

在本节中，你可以找到在项目中使用PCG的最佳实践指南和参考页面。
