**胶囊体阴影（Capsule Shadows）** 可用于启用骨架网格体的柔和阴影支持，工作原理是使用一个基于物理资源打造的角色胶囊体代表物。 柔和阴影能在非直接光照区域中将角色投射到地面，并在直接光照区域中形成柔和阴影。
