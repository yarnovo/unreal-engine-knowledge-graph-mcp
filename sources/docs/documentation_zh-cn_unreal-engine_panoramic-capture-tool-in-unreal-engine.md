Epic Games不再支持或维护全景采集插件。它只在你希望自行创建解决方案时作为参考存在。该插件可能无法正常工作。

借助虚幻引擎的全景采集（Stereo Panoramic Movie Capture）插件可采集静帧或影片，进行立体查看或播放，制造出实时渲染世界场景的假象。以下部分中，我们将学习如何设置并使用全景采集插件，并了解如何查看创建的立体图像。
